    IosReturnUrl

    The URL that the BankID app will try to launch once the BankID app is finished. Set to null to fall back to default Active Login behaviour. Set to empty string to not launch any URL, and instead the BanKID app will ask the user to open the last app. This will only be applied to iOS as Android automatically launches the previous app.
